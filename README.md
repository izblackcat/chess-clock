# Chess Clock

A minimalist, high-performance Chess Clock application built with **React**, **TypeScript**, and **Vite**. Designed to feel like a native mobile app with OLED-friendly dark mode, touch controls, and screen wake-lock capabilities.

## Features

- **Flexible Time Controls:** Presets for Bullet (1+0), Blitz (3+2), Rapid (10+0), and fully **Custom Time/Increment** support.
- **Flip Mode:** Instantly rotate the board 180Â° for face-to-face play without moving the phone.
- **Wake Lock API:** Prevents the screen from sleeping during a game (automatically handled).
- **Fischer Increment:** Correctly handles time bonuses per move.
- **PWA Ready:** Responsive design that feels like a native app on iOS and Android.
- **Modern UI:** Clean typography (JetBrains Mono) and distraction-free dark interface.

## Tech Stack

- **Framework:** React 
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** CSS3 (Variables & Flexbox)
- **Icons:** SVG (Custom lightweight icons)
